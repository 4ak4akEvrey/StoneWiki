# Введение

!!! tip "Прочти это в первую очередь"
    Если вы только зашли на сервер — настоятельно рекомендуем прочитать это введение до конца. Здесь собрано всё, что поможет освоиться в первые часы.

## Команды, которые нужно знать

Запомни эти команды — они открывают весь основной функционал сервера.

| Команда                | Что делает                                                |
| ---------------------- | --------------------------------------------------------- |
| `/menu`                | Общая информация о сервере и быстрый доступ к активностям |
| `/skillmenu` или `/sm` | Меню навыков и их прокачки                                |
| `/quest`               | Ежедневные квесты и квестовые линии                       |
| `/t`                   | Управление городом                                        |
| `/t menu`              | Управление городом через интерфейс                        |
| `/trade <ник>`         | Обмен предметами с другим игроком                         |
| `/ask <вопрос>`        | Любой вопрос по серверу                                   |
| /report <ник>          | Сообщить о нарушении игрока                               |
|                        |                                                           |

??? info "Не запоминай всё сразу"
    Большая часть команд продублирована в `/menu` — открыл интерфейс и тыкай мышкой. Команды нужны когда привыкнешь и захочешь делать всё быстрее.

## Первые 5 часов — режим защиты

!!! warning "Защита временная"
    В течение **первых 5 часов** игры у тебя действует защита от температуры и погодных условий. Ты не будешь замерзать или перегреваться.

За это время желательно успеть **найти место жительства**. Когда защита спадёт — без укрытия и подходящей одежды выжить будет сложнее.

## Чек-лист первого дня

- Прочитал [правила ролевой игры](../main/roleplay.md)
- Открыл `/menu` и осмотрелся
- Вступил в город или нашёл место для базы
- Запустил стартовые квесты через `/quest`
- Прокачал первые навыки в `/skillmenu`
- Нашёл укрытие до окончания защиты от погоды
